# nozaqi-brand-identity

nozaqiのブランドアセットを管理するリポジトリ。

## ディレクトリ構成

```
├── _Fixed/       確定版
└── _Public/      外部公開用
```

## 含まれるアセット

- `avatar.png` - アバター画像（PNG）
- `avatar.jpg` - アバター画像（JPG）
- `avatar-gray.png` - グレースケールアバター
