# nozaqi-brand-identity

nozaqiのブランドアイデンティティアセットを管理するリポジトリ。

## ディレクトリ構成

```
├── Assets/       素材ファイル
├── Source/       Illustratorソースファイル
├── Outputs/      作業中の出力ファイル
├── _Fixed/       確定版（マスターデータ）
└── _Public/      外部公開用（S3に同期）
```

## 含まれるアセット

### Source（Illustratorファイル）
- `wordmark.ai` - ワードマーク
- `tagline.ai` - タグライン
- `platform-headers.ai` - 各プラットフォーム用ヘッダー
- `web-icons.ai` - Web用アイコン

### _Fixed / _Public
- `avatar.png` - アバター画像
- `avatar.jpg` - アバター画像（JPG）
- `avatar-gray.png` - グレースケールアバター
